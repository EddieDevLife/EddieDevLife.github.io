Devinha

Engenheira de Dados & MLOps Autônoma

Dada uma missão, a Devinha ingere dados, valida data contracts, roda detectores de anomalia, registra no MLflow e dispara alarmes via MCP — tudo sozinha. Uma aplicação de referência: o Eye of Sauron, análise de dados públicos contra corrupção. O alerta canônico (risco/hipótese/evidência) é o idioma comum entre os dois.

⚡ MLOps 🏛️ Lakehouse Bronze/Silver/Gold 📜 Data Contracts 📊 MLflow v3 — Em Desenvolvimento
Arquitetura

Como a Devinha Funciona

Um pipeline de agentes autônomos que cobrem todo o ciclo do dado: da ingestão resiliente ao alarme governado.

1. Intake

Normaliza e valida a missão em linguagem natural

→

2. Planner

Decompõe em DAG de tarefas com gates de verificação

→

3. Lakehouse

Ingere dados em Bronze → Silver → Gold (Parquet + DuckDB)

→

4. Verification

Data contracts + policy checks de ética (bloqueia por máquina)

→

5. Alarm

Sink governado via MCP (ticket, dashboard, bot)
