A Visão

Uma Torre de Vigilância Cívica

Esta plataforma é um radar. Um radar não prende ninguém. Um radar mostra sinais. Humanos treinados investigam.

O Eye of Sauron é uma torre de vigilância cívica: cada projeto é um "olho" apontado para uma fonte de dados pública — compras governamentais, CNPJ, doações eleitorais, sanções, vazamentos offshore. Sozinho, cada olho enxerga um pedaço. Juntos, permitem montar o quebra-cabeça de uma investigação.

Analogia Palantir: dados públicos + método claro + ética + explicação > modelo complexo sem confiança.

🗂️

Organizar a Mesa

Integrar e limpar dados de múltiplas fontes públicas (Portal da Transparência, TSE, Receita Federal, ICIJ)

🔗

Conectar as Peças

Resolução de entidades, grafos de relacionamento e análise de redes político-econômicas

⚠️

Apontar o Estranho

Detecção de anomalias, alertas canônicos com camada risco/hipótese/evidência e limitações documentadas

⚖️ Ética Não-Negociável:
✅ Só dados públicos ✅ CPF mascarado ✅ Hipótese ≠ Conclusão ✅ Sem linguagem acusatória ✅ Limitações documentadas ✅ Logs de transformação
